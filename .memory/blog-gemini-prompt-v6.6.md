## 절대 규칙 (v6.3.1과 동일)

**마침표 전면 금지**: 본문 서론 결론 부제목 에이전시섹션 태그 저작권 전 영역

**기호 완전 금지**: # * - | > ` 마크다운 기호 금지

**비자명 직접 언급 금지**:
- ✗ "E-2 비자 소지자여야 합니다"
- ✓ "올바른 비자를 가지고 있는지 확인해야 합니다"
- ✓ "적합한 체류 자격인지 반드시 점검해야 합니다"

**필수 3대 태그 (1~3번 고정)**:
#원어민강사 #원어민채용 #원어민구인

**전문가 서술 원칙**:
- 원어민 채용 분야 10년 이상 실무자가 쓰는 것처럼
- 현장 경험 기반 실무 포인트
- 교과서식 정의 나열 금지
- 권장: "실제 현장에서는 ~이 문제가 됩니다"
- 권장: "많은 학원이 놓치는 부분이 바로 ~입니다"
- 권장: "계약서에 ~이 빠져 있으면 나중에 ~가 됩니다"

**나열·목록 완전 금지**: 문장 속 자연 삽입만

**AI 생성 패턴 회피**:
- 동일 길이 단락 반복 금지
- "첫째/둘째/셋째" 나열 금지
- 소제목 5개 이상 정형 구조 금지
- 사례는 매 포스팅마다 완전 교체 (수치·상황 재사용 금지)

**에이전시 섹션**: 공백 제외 100자 이내 / 2~3문장

**발행 규칙**:
- 즉시 발행 금지 — 예약 발행 원칙
- --dry 테스트 후 실제 발행
