  - Install Now 클릭

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

3단계. USB에서 파일 복사
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  - 이 PC(메인)에서:
    Q:\Claudework\bridge base\deploy\craig-portable\package
    이 폴더를 USB에 복사

  - 새 PC에서:
    USB의 package 폴더를 바탕화면에 임시로 붙여넣기

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

4단계. 설치 실행
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  - package 폴더 안의 setup.ps1 파일 찾기
  - setup.ps1 우클릭 → "PowerShell에서 실행"

  ※ 만약 "실행 정책" 에러 나오면:
    1) 시작 메뉴 → "PowerShell" 검색 → 관리자로 실행
    2) 아래 명령어 붙여넣기:
       powershell -ExecutionPolicy Bypass -File "바탕화면\package\setup.ps1"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

5단계. 설치 중 입력할 것
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  - 설치 경로: Enter (기본 D:\BridgeCraig)
  - Craigslist Email: (계정 이메일 입력)
  - Craigslist Password: (비밀번호 입력 - 화면에 안 보임)
  - City: Enter (기본 seoul)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

6단계. 완료 확인

  - 더블클릭 → 바로 광고 자동 게시 시작
  - 6시간마다 자동 실행됨 (PC 켜져있을 때)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【사용법】

  밥 먹으러 갈 때:
    → 바탕화면 "BRIDGE Craig RPA" 더블클릭
    → 알아서 돌아감 (완료 후 자동 닫힘)

  로그 확인:
    → D:\BridgeCraig\logs\scheduler.log

  새 채용공고 업데이트:
    → 메인 PC에서 pack.ps1 재실행 → USB 복사 → setup.ps1 재실행

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【삭제 방법】

  PowerShell에서:
    schtasks /Delete /TN "BridgeCraigslistRPA" /F
    → D:\BridgeCraig 폴더 삭제
    → 바탕화면 아이콘 삭제

  - 로그인 완료되면 터미널로 돌아옴
  - 바로 대화 시작 가능!

D단계. 프로젝트 연결 (선택)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Craig RPA만 할 거면 이 단계 필요 없음.
  Bridge 프로젝트 전체 작업하려면:

  1) Git 설치: https://git-scm.com/downloads
  2) PowerShell에서:
     git clone https://github.com/koreadobby/bridge.git
     cd bridge
     claude

  → 이러면 나(Claude)가 프로젝트 전체를 보면서 작업 가능

【요약: 새 PC에서 전부 설치하는 순서】

  1. Python 설치  (python.org)
  2. Chrome 확인  (google.com/chrome)
  3. Node.js 설치 (nodejs.org)
  4. PowerShell에서: npm install -g @anthropic-ai/claude-code
  5. PowerShell에서: claude  → 로그인
  6. USB에서 setup.ps1 실행 → Craig RPA 설치
  7. 바탕화면 "BRIDGE Craig RPA" 더블클릭 → 완료!
