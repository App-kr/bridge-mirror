## ⚠️ 응급 복구 (필요시)

### RDP 포트를 3389로 복구
```powershell
# 레지스트리 복원
Set-ItemProperty -Path 'HKLM:\System\CurrentControlSet\Control\Terminal Server\WinStations\RDP-Tcp' -Name 'PortNumber' -Value 3389 -Force

# 방화벽 규칙 삭제
netsh advfirewall firewall delete rule name='RDP-Secure-4389'

# 기본 RDP 규칙 재활성화
netsh advfirewall firewall set rule name='Remote Desktop - User Mode (TCP-In)' new enable=yes
```
